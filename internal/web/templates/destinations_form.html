How to generate this API key {{if eq $dt "emby"}}
  1. Open Emby's web UI as an administrator.
  2. Go to Settings (gear icon) → Server → Advanced → API Keys.
  3. Click + New API Key. Give it a name like "Audplexus".
  4. Copy the generated key. Emby may show it only once depending on version.
{{else if eq $dt "jellyfin"}}
  1. Open Jellyfin's web UI as an administrator.
  2. Go to Dashboard → API Keys (under Advanced).
  3. Click +. Give it an app name like "Audplexus".
  4. Copy the generated key. Jellyfin shows it once at creation.
{{else if eq $dt "abs"}}
  1. Open Audiobookshelf as an administrator.
  2. Go to Settings (gear icon) → Users.
  3. Click your admin user.
  4. Scroll to API Keys → Generate API Key. Required scope: admin.
  5. The token is shown once at generation. Copy it immediately and paste here.
{{end}}

How to find your library ID {{if eq $dt "emby"}}
  1. In a terminal, run: curl 'http://<emby>:8096/emby/Library/MediaFolders?api_key=<your_key>' | jq
  2. Find the entry where CollectionType is audiobooks.
  3. Copy its ItemId field. That's your library ID.
{{else if eq $dt "jellyfin"}}
  1. In a terminal, run: curl -H 'X-Emby-Token: <your_key>' 'http://<jellyfin>:8096/Library/VirtualFolders' | jq
  2. Find the entry with "CollectionType": "books" and an audiobook path.
  3. Copy its ItemId field. That's your library ID.
{{else if eq $dt "abs"}}
  1. In a terminal, run: curl -H 'Authorization: Bearer <your_key>' 'http://<abs>:80/api/libraries' | jq '.libraries[] | {id,name,mediaType}'
  2. Find the entry where mediaType is book (your audiobook library).
  3. Copy its id field (a UUID like 00d44f08-df07-...). That's your library ID.
  4. UI alternative: open Audiobookshelf web → click your audiobook library → the URL /library/<id> contains it.
{{end}}
